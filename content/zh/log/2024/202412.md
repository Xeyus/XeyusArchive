<!-- node 2024.12.04 -->

雅思比四级难多了。

<!-- node 2024.12.08 -->

去健身了。准确来说叫复建。有点累。

今天查卫生。19 点多时男寝被查，19:39 人还没到女寝。

<!-- node 2024.12.09 -->

下周开始就能上滑冰课了。一周五节，非常好。

<!-- node 2024.12.10 -->

昨天扒开的橘子到了今天也才吃了几瓣，用于开胃消食。刚一入口，是类似纯白糖的甜味，随后就是无止境的酸，踩着一丝若有若无的齁甜。

<!-- node 2024.12.13 -->

我抹的不是身体乳而是防晒霜？

好吧，反正差不多。

<!-- node 2024.12.14 -->

四级考试，但答完就睡。没意思。

花园城的熊喵来了明天闭店。虽然隔壁也有，但是花园城直连地铁口，去这边这家更快。

<!-- node 2024.12.15 -->

今天 7:50 左右醒了，玩一会儿接着睡。

醒的原因一是生物钟，二是做梦梦见敲了心理课但是和老师解释时说“我确实不该不去思想政治课”。

<!-- node 2024.12.17 -->

今天，冰场还没好，上的轮滑课。生物那边估计是换教务处的人了，把课排到了体育课的时候（不知道滑冰现在才开始上），属于教学事故。由于体委在生物的人里，新选了个体委。

我感觉此人有些眼熟。

老师教了腰部和腿部应如何向侧边发力，调整重心，并讲了正确的发力会有什么样的感觉（腿越来越酸，但是换腿后立刻就放松了，不过腰会疼）。利用这个性质和物理学原理，我成功地滑走了，而且滑得很快。

然后我明白体委眼熟在哪里了。就他一个人还在试图向前“走”，以走路的方式向前蹭然后往后出溜。是的，他就是上节课结束时抱着柱子听老师宣布下课的人之一。

发现自己滑起来时就一条腿正常发力，询问老师，老师表示这就像骑自行车，先骑熟了，骑起来了，才能讲究别的。于是在再次经过体委旁边时提醒他向侧边发力。

最后他还是偏向“走”，不过能稳当一些了。希望他滑冰时不会趴冰上起不来。

<!-- node 2024.12.18 -->

梯子突然不好使了。交叉求证并修复一切可能的问题后，仍然连不上外网。怀疑人生时顺手点了下 outlook，也登不上去。

看来只是由于局势问题暂时把墙加厚了，不是我电脑技术不行。太好了。

<!-- node 2024.12.19 -->

现在梯子就好使了。果然没啥问题。

不过现在痛经有点严重。不知道为什么疼得厉害。可能是因为我这几天没吃冰淇淋？之前几次都吃了来着。

<!-- node 2024.12.24 -->

博德之门 3 真好玩。暂时停更。但是肯定还在写东西，只不过更得慢了。恢复正常更新估计得等到 2025.1.15。

<!-- node 2024.12.26 -->

同学把自行车的钥匙丢了，不好意思回去撬锁。我提出主意：再上把锁，一起撬，装作被无耻之人多上了一把。或者干脆把这片所有自行车都锁上。这样就不会有人追究你撬锁的事了。
