<!-- node 2025.06.01 -->

跑团，相当累但是有趣。

<!-- node 2025.06.08 -->

这次跑团跑一半就散了。书香社需要我们过去当部长副部长，具体事宜下周日再说。

<!-- node 2025.06.10 -->

自己独立完成了机器人实验（虽然只是装驱动看进程创立工作区写程序而已），但我没想到有人照着抄代码都能抄出满屏报错。

<!-- node 2025.06.11 -->

工程制图 3.5，很好，没啥问题，过了。再也不想手绘图样了。思想政治实践课 3.0，不清楚它是怎么给绩点的，但是我觉得在自己差不多可以确定 ASD 的情况下，这种课还能拿到这个绩点已经很好了，和我体育能拿到 3.7 一个道理。

而且那种清晰的梦境要回来了。虽然还是有些模糊，有些平淡（说真的，跑团跑出和粮食有关的克苏鲁问题已经算平淡的了，更别提去想塞尔维亚旅游但是所有道路都绕回了原点），但终究是能较清晰地记住，以及体会其中的过程了。更好的是，我还能更清晰地觉察出，这是梦/这不合理，并控制它循环（读档）。

<!-- node 2025.06.21 -->

酸汤鱼很好吃。但是可能有我的过敏源，导致我半夜犯了延迟过敏症状。或者不是延迟过敏，而是荔枝过敏。

打牙砂锅的肘子非常好吃入味，小白菜汤疑似是用肘子汤做的，也很好吃。

<!-- node 2025.06.22 -->

又去吃了顿火锅。应该是因为中午没吃饭，所以一盘肉 + 一篮菜 + 虾滑还有面都吃完了。这里可比学校食堂好多了。

<!-- node 2025.06.25 -->

虽然好几天没做电路原理题，但是再做到老师发的期末复习题时还是没太生疏，基本上都没问题——除了验证结果时发现还得是 Chat GPT 和 Deepseek，不论是它们最新的模型还是普通模型，都在数理逻辑方面吊打别家 AI。有的就是我讲多少遍听不明白，有的，比如它们，就是一点就通（如果出错了的话）还能自行总结，甚至表现得我才是老师，它们只是捧场的。

去吃火锅路上还口算了一会儿（25 道）第九章高数题。

算到最后算不下去整个思路就看答案了，因为我早上起来后一直懒得吃饭，饿得看服务员都像积分区间了。于是点了两盘肉，一份虾滑，一碗茼蒿菜和一份面条凑锅底优惠券使用门槛。最后不出意外几乎全吃完了，只剩下 1/3 的面条。相比之前两荤一素一主食，主食必剩否则就吃撑而言，今天确实是饿得急眼了。

同样能证明我确实饿了的是我接下来买了块蛋糕，只有在我当天确实几乎啥也没吃，而且特别需要补充能量时，我才会想买这种甜腻的东西，不然饿了不想吃饭我顶多整瓶碳酸饮料，或是不加糖的奶茶（然后挺到下一个饭点它们还没喝完）。这家蛋糕用料好，不腻，还配了保冷剂，我校园跑回来，又洗了澡，它还是凉丝丝的。所以尽管这家比较贵，我还是会在想吃时选择这里的点心。

<!-- node 2025.06.26 -->

没想到因为老师在考试时一个劲叨叨，感觉外界节拍和脑内节拍不一致进而惊恐发作，呼吸性碱中毒了。不过卷纸确实不怎么难。

<!-- node 2025.06.27 -->

因为还没太缓过来，起来后继续躺了一会儿，翘了机器人概论（反正我作业都交上去了，而且它也不点名）去图书馆练习高等数学。一楼自习室的椅子很舒服，台灯也不错，电源离得很近，很适合学习。最后，25 分钟做 25 道题，正确率 80%，满意了。

晚上去吃了酸汤鱼。还是那么好吃。为了防止意外，吃之前还吃了片抗敏药。距离吃饭过了大约 5 小时后还是没有问题，上次大概这个时候就开始有症状了。

<!-- node 2025.06.29 -->

出去吃自助，还逛了一家唱片店。

“可能吧。”
