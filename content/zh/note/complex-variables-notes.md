- 由某复数不等式/等式确定的复平面区域：

> [!Quote]-
> **1\. 把条件转成实部、虚部形式**设  $z = x + iy$ 。
>
> * 如果条件涉及  $|z|$ ，就转成  $\sqrt{x^2+y^2}$ 。
> * 如果条件涉及  $\operatorname{Re}(z)$ ，就转成  $x$ 。
> * 如果条件涉及  $\operatorname{Im}(z)$ ，就转成  $y$ 。
>   这样就能得到普通的平面几何条件。
>
> ---
>
> **2\. 常见类型归纳**
>
> 1. **模长类条件**
>    * $|z| = r$  → 圆周
>    * $|z| < r$  → 圆的内部
>    * $|z| > r$  → 圆的外部
> 2. **实部/虚部类条件**
>    * $\operatorname{Re}(z) = a$  → 直线  $x=a$
>    * $\operatorname{Im}(z) = b$  → 直线  $y=b$
>    * $\operatorname{Re}(z) > a$  → 半平面
>    * $\operatorname{Im}(z) < b$  → 半平面
> 3. **差的模长类条件**
>    * $|z - z_0| = r$  → 以  $z_0$  为圆心的圆
>    * $|z - z_1| = |z - z_2|$  → 连接  $z_1, z_2$  的线段的垂直平分线
>    * $|z - z_1| < |z - z_2|$  → 平面上靠近  $z_1$  的一边
> 4. **分式/商类条件**
>    * $\left|\frac{z-z_1}{z-z_2}\right| = k$  → 阿波罗尼圆
>    * $\left|\frac{z-z_1}{z-z_2}\right| < 1$  → 平面上距离  $z_1$  更近的区域
>
> ---
>
> **3\. 判断方式总结**
>
> * **先转成  $x,y$  不等式**，看是直线、圆还是其他曲线。
> * **注意对称性**：模长条件一般是圆/圆环，实部虚部条件一般是直线/半平面。
> * **复杂条件**（如分式）往往是圆或圆弧，因为复平面保角变换能把直线与圆互相对应。
>
> ---
>
> **单连通**：区域内部没有“洞”。形式上：任意闭曲线都能缩到一点而不离开区域。
> **多连通**：区域有“洞”，至少有一个封闭曲线不能收缩到一点。
