### 思路

根据求解/证的对象选择方法。先看形式、性质再找可能的解。

证明题不会的话就顺着性质往下推，大概率能推出来。

### 题型

#### 求极限

1. 通过无穷小的代换（仅乘除时可换），因式分解等化简。可以考虑求导算极限。
2. 观察是否为 $\frac{0}{0}$（或 $\frac{∞}{∞}$），若是或者能化成这种形式，就洛必达。
   1. 想接着洛时看看是不是符合形式；
   2. ∞*0: 其中一个变分母；
   3. $∞-∞$: 恒等变形，通分；
   4. $0^0$，$∞^0$，$1^∞$: 幂指函数通用的 e 的对数次幂化简；
   5. 其中 $1^∞$ 有公式，极限那里说过，不写了。打着太麻烦。
3. 若不是，可使用极限的四则运算，比较无穷小阶数等。
4. 若还不行，若为分式，按照最高次泰勒展开；或者使用夹逼定理（常见于类似数列形式的求极限）。
   1. 麦克劳林：$x_0=0$ 的泰勒。
5. 以及最后，最基础的ε 。
6. 注意是否为单侧极限，是否有极限。

#### 求导与微分

1. 直接考察求导和微分：
   1. 直接求导/微分。一元函数求导=微分。
   2. y=f(x) 的反函数求导/微分：将 x 视作关于 y 的函数。$\frac{dx}{dy}=\frac{1}{\frac{dx}{dy}}$。
   3. 隐函数求导/微分：两端对 x 求导，y 视作关于 x 的函数。
   4. 乘除与幂指：两端取对数。
   5. 参数方程：两端取微，多阶则视作 (n-1) 阶和 x=φ(x) 的参数方程再次两端取微。
   6. 过高阶导数找规律。
2. 应用题：
   1. 相关变化率：微分思想。
   2. 求近似值：运用微分的定义公式。
   3. 其他，就题论题。
3. 是否可导：
   1. 连续不一定可导/可微，可导/可微一定连续。
   2. 看左右导数是否存在且相等。

##### 连续和间断

第一类间断点：可去间断点，仅该点无定义；跳跃间断点，该点两侧存在不同定义。

第二类间断点：振荡间断点，该点附近无限振荡；无穷间断点，该点至少一侧的极限趋于无穷。均为极限不存在。

#### 求零点/f(x)=y 在 [a,b] 的根个数

1. 先看函数性质。
2. 观察其加减、附加系数、变量确定其“原函数”。
3. 单看它不行就用罗尔、柯西、拉格朗日中值定理。
   1. 一个函数，区间端点函数值相等，区间内可导 -->罗尔；
   2. 两个函数/一个分式，可以凑出 $\frac{f(a)-f(b)}{g(a)-g(b)}$-->柯西；
   3. 一个函数在两点的函数值作差 -->拉格朗日。

### 积分

#### 基本概念

1. 不定积分：如果 F′(x)=f(x)，则 F(x)+C 是 f(x) 的不定积分，记作：$\int f(x)dx=F(x)+C$
   物理意义：不定积分可以看作是累积变化的反函数。
2. 定积分：定积分是曲线下方面积的代数和，用来表示函数在区间 [a,b] 上的累积变化量。$\int_{b}^{a} f(x)dx$
   几何意义：定积分表示曲线 y=f(x) 和 x 轴之间的有符号面积。
   根据牛顿 - 莱布尼茨公式：$\int_{b}^{a} f(x)dx$=F(a)-F(b)

#### 基本性质

1. 线性性质：对定积分和不定积分成立。
   线性运算：加减和数乘。
2. 区间可加性：对定积分成立，区间 [a,b]=[a,c]+[c,b]。同理区间颠倒，值取相反数；规定区间上下限相等，定积分值为 0。对有限成立。
3. 奇偶性：
   1. 若 f(x) 为奇数，积分区间关于原点对称，积分为 0——化简被积函数
   2. 若 f(X) 为偶数，积分区间关于原点对称，$\int_{-a}^{a} f(x)dx=2\int_{0}_{a} f(x)dx$——缩小积分区间
   3. 周期函数任一周期 -->[0,T]
4. 被积函数较大，则积分较大。
5. 任何连续函数都存在原函数。

### 向量代数与解析几何

#### 向量

**内积（点积）**
算法：对应分量相乘后求和，或模长乘余弦夹角。
适用条件：同维向量（如二维/三维）。
几何意义：投影长度乘积，或判断两向量垂直（内积为 0）。

**外积（叉积）**
算法：三维行列式计算，方向由右手法则确定，模长为两向量模长乘正弦夹角。
适用条件：仅三维向量。
几何意义：结果垂直原向量，模长表平行四边形面积，或判断平行（外积为 0 向量）。-->垂直关系如平面内直线方向向量与法向量。

**混合积**
算法：先计算两向量的外积，再与第三向量点乘，结果为标量。
适用条件：三个三维向量。
几何意义：绝对值表平行六面体体积，混合积为 0 则三向量共面。
可判断两直线空间位置关系 --$s_1xs_2·r$，若混合积为 0 则共面，否则异面

#### 平面

1. 平面方程：点法式，一般方程，截距式。
2. 点到平面距离公式：（类似点到直线距离公式）-->平行平面距离：$\frac{|D_2-D_1|}{\sqrt{A^2+B^2+C^2}}$
3. 利用数量积求平面夹角：（不是这高中天天用啊，法向量内积比模积）
4. 空间直线一般方程：两平面交线；
5. 直线点向式和参数方程：可两两化作参数方程，分母为零也没关系，对应参数=（p-q）的 q；一般方程：法向量外积=方向向量，令一个变量为 0 解出一个在直线上的点，转为点向式。
6. 直线夹角：（似曾相识的方向向量内积比模积）
7. 直线与平面夹角：（似曾相识的 sinθ=内积比模积，对了前面两个求都是 cosθ，原因易得）
8. 异面直线距离：找公垂线易得，设线上任取两点得一向量*r*，则 $d=\frac{|s_1xs_2·r|}{|s_1xs_2|}$
9. 平面束：为方便计算，经常只取一个变量，丢了的那个平面另讨论。
10. 直线在平面上的投影直线：使用平面束计算，寻找和已知平面垂直的平面与它的交线。其他线 - 面题目也用平面束（不然基本找不到联系）。
11. 另外，坐标轴投影形成的四面体体积为 $\frac{1}{6}xyz$

#### 点与直线

1. 通用：找方向向量与直线上一点；
2. 平面过点与直线：方向向量与点 - 直线上一点做外积可得平面法向量；
3. 点到空间直线距离：
设点 $\( P(x_0, y_0, z_0) \)$，直线由点 $\( Q(x_1, y_1, z_1) \)$ 和方向向量 $\( \mathbf{v} = (a, b, c) \)$ 确定：
$d = \frac{\lVert (\mathbf{Q} - \mathbf{P}) \times \mathbf{v} \rVert}{\lVert \mathbf{v} \rVert}$
其中：
$\( \mathbf{Q} - \mathbf{P} = (x_1 - x_0, y_1 - y_0, z_1 - z_0) \)$
4. 点与直线、平面的相关点同理。只不过大部分需要找到垂直点（过 P 垂直于直线的面与直线交点）再变换。
5. 线与面交点：代入求解。

#### 曲面方程

空间曲线由两曲面交线构成，那么该曲线的切向量一定同时平行于两个曲面的法向量的叉积。

##### 柱面

易得，缺一个变量的基本上都是柱面（一曲线在另一维度无限延伸）。

以及曲面方程可由曲线方程推广学习理解。

##### 二次曲面

**椭球面**

方程：x²/a² + y²/b² + z²/c² = 1。

几何意义：三维闭合对称曲面，三轴拉伸的球体。

判别：二次项系数同号，常数项为正。

**单叶双曲面**

方程：x²/a² + y²/b² - z²/c² = 1。

几何意义：单侧无限延伸曲面，形似双曲线旋转体。

判别：两正一负系数，常数项为正。

**双叶双曲面**

方程：-x²/a² - y²/b² + z²/c² = 1。

几何意义：分离的两片曲面，沿主轴方向开口。

判别：两负一正系数，常数项为正。

***椭圆抛物面**

方程：z = x²/a² + y²/b²。

几何意义：碗状开口向上，横截面为椭圆。

判别：二次项系数同号，一次项存在。

**双曲抛物面**

方程：z = x²/a² - y²/b²。

几何意义：马鞍形，双向相反开口。

判别：二次项系数异号，一次项存在。

**二次锥面**

方程：x²/a² + y²/b² - z²/c² = 0。

几何意义：双锥面，顶点在原点，沿轴无限扩展。

判别：二次项系数不全同号，常数项为零。

##### 旋转曲面

1. 旋转曲面通用求法：旋转一周=圆=圆周到旋转轴对应点距离相等，由此可得原 m=f(n)-->a=F(b,c)。
2. 所以有变量替换规则：
   绕 X 轴旋转：将原方程中的非旋转轴变量（如 y）替换为 $\sqrt{y^2 + z^2}$，适用于平面曲线在 XY 或 XZ 平面上。
   绕 Y 轴旋转：将原方程中的非旋转轴变量（如 x）替换为 $\sqrt{x^2 + z^2}$，适用于平面曲线在 XY 或 YZ 平面上。
   绕 Y 轴旋转：将原方程中的非旋转轴变量（如 z）替换为 $\sqrt{x^2 + y^2}$，适用于平面曲线在 XZ 或 XZ 平面上。
3. 若圆锥曲线不在 XY、XZ 或 YZ 平面，需先将曲线平移或旋转到标准平面，再应用上述替换规则。例如：
   1. 平面变换：通过坐标系旋转或平移，将原曲线方程转换到 XY/XZ/YZ 平面。
   2. 应用旋转规则：按上述替换规则生成旋转曲面方程。
   3. 逆变换：将生成的曲面方程还原到原坐标系。
4. 对于任意平面内的圆锥曲线绕任意轴旋转，可通过参数方程和旋转矩阵推导：
   1. 原曲线方程化为 $r(t)=(x(t),y(t),z(t))$
   2. 绕某轴旋转角θ，用旋转矩阵 $R_?(θ)$ 作用，此处以绕 z 轴为例：$x'=x(t)cosθ-y(t)sinθ$, $y'=x(t)sinθ+y(t)cosθ$, $z'=z(t)$
   3. 消去参数 t 和θ，得到 $F(x,y.z)=0$
5. 由上易得：旋转椭圆、双曲线：绕 a，缺 b，随 c。

### 多元函数微分

#### 求偏导数

对某一元求导，方法同一元函数求导。判断存在即判断是否所有方向的方向导数都存在，可以举反例或用极坐标算。

偏导数存在且连续 ⇒ 全微分存在（充分条件）。

全微分存在 ⇒ 偏导数必存在，但未必连续（必要条件）。

隐函数存在定理：F(???)=0, 该点邻域内连续且可微，偏导数不为零的可以确定一个隐函数。

对于 $f(x,y)$，记：

$$
D = \det(H) = f_{xx} f_{yy} - (f_{xy})^2,
$$

则：

$D > 0$ 且 $f_{xx} > 0$ → 极小值。

$D > 0$ 且 $f_{xx} < 0$ → 极大值。

$D < 0$ → 鞍点。

$D = 0$ → 需更高阶导数或其他方法。

#### 全微分计算

$dz=\frac{∂z}{∂x}dx+\frac{∂z}{∂y}dy$

计算近似值：改写为 x+dx,y+dy，之后求 z(x,y) 偏导数得全微分，z(x,y)+dz 可得近似值。

#### 方向导数与梯度

方向导数∇f⋅u，方向向量乘该点对应方向偏导数。

梯度 $∇f=(\frac{∂f}{∂x},\frac{∂f}{∂y})$，方向为最大方向导数方向。

### 重积分

重积分是对多元函数在某一区域上的积分。

二重积分：二元函数 $f(x, y)$ 在平面区域 $D$ 上的积分，几何意义为曲顶柱体体积。

三重积分：三元函数 $f(x, y, z)$ 在空间区域 $\Omega$ 上的积分，物理意义可用于计算质量等。

运算方法

累次积分：

二重积分：化为两次单变量积分（先 $x$ 后 $y$ 或反之），需确定积分限（X/Y 型区域）。

三重积分：化为三次积分（如直角坐标下 $z \rightarrow y \rightarrow x$），或投影到坐标面后分层积分。

坐标变换：

极坐标（二重积分）：适合圆形区域，$dA = rdrd\theta$。

柱坐标（三重积分）：$dV = rdrd\theta dz$，适用于轴对称区域。

球坐标（三重积分）：$dV = \rho^2 \sin\phi d\rho d\phi d\theta$，适合球对称区域。

需乘雅可比行列式（如极坐标中 $r$）。

简化技巧

对称性：

区域对称且被积函数具奇偶性时，积分可简化为部分区域积分的倍数，或直接为零。

变量分离：若被积函数可分解为单变量函数乘积，逐次积分更简便。

积分次序交换：简化积分限或积分难度（如 Fubini 定理）。

应用

几何应用：

二重积分求平面区域面积或空间体积；三重积分求空间体积。

物理应用：

质量（密度函数积分）、质心坐标、转动惯量、引力、电场强度等。

转动惯量：

1. **直角坐标系**：
   - 绕 $x$ 轴的转动惯量：
     $I_x = \iint\limits_{D} y^2 \rho(x, y) \, dx \, dy$
   - 绕 $y$ 轴的转动惯量：
     $I_y = \iint\limits_{D} x^2 \rho(x, y) \, dx \, dy$
   - 绕 $z$ 轴（垂直平面）的转动惯量：
     $I_z = \iint\limits_{D} (x^2 + y^2) \rho(x, y) \, dx \, dy$
2. **极坐标系**（简化对称情况）：
   - 绕 $z$ 轴的转动惯量：
     $I_z = \iint\limits_{D'} r^3 \rho(r, \theta) \, dr \, d\theta$
     其中 $x^2 + y^2 = r^2$，面积微元 $dA = r \, dr \, d\theta$
注：$\rho(x, y)$ 或 $\rho(r, \theta)$ 为面密度函数，$D$ 或 $D'$ 为积分区域。

质心：

总质量 $M$：

$$M = \iint\limits_{D} \rho(x, y) \, \mathrm{d}A$

质心坐标：
$\bar{x} = \frac{1}{M} \iint\limits_{D} x \rho(x, y) \, \mathrm{d}A$，
$\bar{y} = \frac{1}{M} \iint\limits_{D} y \rho(x, y) \, \mathrm{d}A$

或合并表示为：
$\bar{x} = \frac{\iint\limits_{D} x \rho(x, y) \, \mathrm{d}A}{\iint\limits_{D} \rho(x, y) \, \mathrm{d}A}$，
$\bar{y} = \frac{\iint\limits_{D} y \rho(x, y) \, \mathrm{d}A}{\iint\limits_{D} \rho(x, y) \, \mathrm{d}A}$$

概率统计：联合概率密度函数的概率计算。

平均值：函数在区域上的平均值为积分除以区域度量（面积/体积）。

### 曲线积分与曲面积分

好的，理解曲线积分和曲面积分的核心在于把握它们的物理意义、几何意义以及计算方法的本质。下面用最简洁清晰的语言分点列出关键知识：

核心概念：积分域的升维

定积分 (一重积分)： 积分域是 直线段 (一维)。计算函数在直线段上的累积量（如长度、质量、功 - 如果力方向沿直线）。

二重积分： 积分域是 平面区域 (二维)。计算函数在平面区域上的累积量（如面积、质量、电荷面密度）。

三重积分： 积分域是 空间区域 (三维)。计算函数在空间区域上的累积量（如体积、质量、电荷体密度）。

曲线积分： 积分域是 空间或平面中的曲线 (一维)。计算函数在曲线上的累积量。

曲面积分： 积分域是 空间中的曲面 (二维)。计算函数在曲面上的累积量。

曲线积分 (Line Integrals)

核心思想： 在一条弯曲的路径（曲线）上进行积分。

分类：

第一类曲线积分 (对弧长的曲线积分)：

物理意义： 计算标量场 (如线密度 f(x, y, z)) 沿曲线 C 的累积总量（如曲线的质量、电荷量）。

几何意义： 曲线自身的属性累积（与方向无关）。

记号： ∫_C f(x, y, z) ds

关键： ds 是曲线的弧长微元。积分结果是一个标量。

计算方法 (核心步骤)：

参数化曲线： 将曲线 C 表示为参数方程 r(t) = (x(t), y(t), z(t)), t ∈ [a, b]。

计算弧长微元： ds = ||r'(t)|| dt = √[ (dx/dt)² + (dy/dt)² + (dz/dt)² ] dt

代入积分： ∫_a^b f( x(t), y(t), z(t) ) * √[ (dx/dt)² + (dy/dt)² + (dz/dt)² ] dt

第二类曲线积分 (对坐标的曲线积分)：

物理意义： 计算向量场 F(x, y, z) = (P(x, y, z), Q(x, y, z), R(x, y, z)) 沿有向曲线 C 所做的功，或向量场沿曲线的环量/流量。

几何意义： 向量场与曲线切向量的点积累积（与方向强相关）。

记号： ∫_C P dx + Q dy + R dz 或 ∫_C F · dr (其中 dr = (dx, dy, dz) 是切向量微元)

关键： dx, dy, dz 是坐标的微分。积分结果是一个标量。方向至关重要：反转曲线方向，积分值变号。

计算方法 (核心步骤)：

参数化曲线 (带方向)： r(t) = (x(t), y(t), z(t)), t 从 a 到 b 对应曲线方向。

求导得切向量微元： dr = r'(t) dt = (dx/dt dt, dy/dt dt, dz/dt dt) = (dx, dy, dz)

点积并积分： ∫_a^b F( r(t) ) · r'(t) dt = ∫_a^b [ P(x(t), y(t), z(t)) (dx/dt) + Q(x(t), y(t), z(t)) (dy/dt) + R(x(t), y(t), z(t)) (dz/dt) ] dt

格林公式 (Green's Theorem) (平面曲线)：

联系： 建立了平面闭曲线 C (正向，逆时针) 上的第二类曲线积分与其所围平面区域 D 上的二重积分之间的关系。

公式： ∮_C P dx + Q dy = ∬_D (∂Q/∂x - ∂P/∂y) dA

意义： 将复杂的曲线积分转化为相对容易的区域积分（或反之）。揭示了向量场在区域上的旋度 (∂Q/∂x - ∂P/∂y) 与其沿边界环量之间的关系。

曲面积分 (Surface Integrals)

核心思想： 在一个弯曲的“薄片”（曲面）上进行积分。

分类：

第一类曲面积分 (对面积的曲面积分)：

物理意义： 计算标量场 (如面密度 g(x, y, z)) 在曲面 S 上的累积总量（如曲面的质量、电荷量）。

几何意义： 曲面自身的属性累积（与方向无关）。

记号： ∬_S g(x, y, z) dS

关键： dS 是曲面的面积微元。积分结果是一个标量。

计算方法 (核心步骤)：

参数化曲面： 将曲面 S 表示为参数方程 r(u, v) = (x(u, v), y(u, v), z(u, v)), (u, v) ∈ D (参数域)。

计算法向量微元： 求偏导数 r_u = ∂r/∂u, r_v = ∂r/∂v。面积微元 dS = ||r_u × r_v|| du dv。

代入积分： ∬_D g( x(u, v), y(u, v), z(u, v) ) * ||r_u × r_v|| du dv

第二类曲面积分 (对坐标的曲面积分)：

物理意义： 计算向量场 F(x, y, z) = (P, Q, R) 穿过有向曲面 S 的通量 (Flux) (如流体流过曲面、电场穿过曲面)。

几何意义： 向量场与曲面法向量的点积累积（与方向强相关）。

记号： ∬_S P dy dz + Q dz dx + R dx dy 或 ∬_S F · dS (其中 dS = n dS 是有向面积微元，n 是单位法向量)。

关键： 必须指定曲面的侧 (方向) (如外侧、上侧)。反转法线方向，积分值变号。

计算方法 (核心步骤 - 常用参数化法)：

参数化曲面 (带方向)： r(u, v) = (x(u, v), y(u, v), z(u, v)), (u, v) ∈ D。

计算法向量： r_u × r_v。根据指定的侧，确定其方向（或单位化 n = (r_u × r_v) / ||r_u × r_v||）。

点积并积分： ∬_S F · dS = ∬_S F · n dS = ∬_D F( r(u, v) ) · (r_u × r_v) du dv

或者，如果曲面能表示为 z = z(x, y) (上侧)：

∬_S F · dS = ∬_D [ -P(x,y,z(x,y)) ∂z/∂x - Q(x,y,z(x,y)) ∂z/∂y + R(x,y,z(x,y)) ] dx dy

高斯公式 (Gauss's Theorem / Divergence Theorem)：

联系： 建立了空间闭曲面 S (外侧) 上的第二类曲面积分与其所围空间区域 Ω 上的三重积分之间的关系。

公式： ∯_S F · dS = ∭_Ω div F dV (其中 div F = ∂P/∂x + ∂Q/∂y + ∂R/∂z 是向量场 F 的散度)

意义： 将复杂的曲面积分转化为相对容易的区域积分（或反之）。揭示了向量场在区域上的散度 div F (源/汇强度) 与其穿过边界的净通量之间的关系。

斯托克斯公式 (Stokes' Theorem)：

联系： 建立了空间有向曲面 S (边界 C 的方向由右手法则确定) 上的第二类曲面积分与其边界曲线 C 上的第二类曲线积分之间的关系。

公式： ∬_S curl F · dS = ∮_C F · dr (其中 curl F = (∂R/∂y - ∂Q/∂z, ∂P/∂z - ∂R/∂x, ∂Q/∂x - ∂P/∂y) 是向量场 F 的旋度)

意义： 将复杂的曲面积分转化为相对容易的曲线积分（或反之）。揭示了向量场在曲面上的旋度 curl F (旋转强度) 与其沿边界环量之间的关系。

总结关键点

积分域： 曲线积分在曲线 (1D) 上，曲面积分在曲面 (2D) 上。

两类积分：

第一类 (标量场)： 积分函数值 × 长度/面积微元 (ds / dS)。结果标量，无方向性。计算核心是参数化 + 求微元长度/面积。

第二类 (向量场)： 积分向量场 · 有向切向量/法向量微元 (dr / dS = n dS)。结果标量，有强方向性 (曲线方向/曲面侧)。计算核心是参数化 + 求 dr 或 r_u × r_v + 点积。

参数化是基石： 无论是曲线还是曲面积分，第一步几乎总是找到合适的参数方程 r(t) 或 r(u, v)。

三大公式 (核心联系)：

格林： 平面闭曲线积分 (第二类) ↔ 二重积分 (区域上旋度)。

高斯： 空间闭曲面积分 (第二类) ↔ 三重积分 (区域上散度)。

斯托克斯： 空间曲面积分 (第二类) ↔ 空间曲线积分 (第二类) (曲面旋度 ↔ 边界环量)。

物理意义驱动理解：

第一类曲线积分：曲线质量。

第二类曲线积分：力场做功。

第一类曲面积分：曲面质量。

第二类曲面积分：向量场通量。

学习建议

不要死记公式！ 理解物理意义和几何意义是核心。

掌握参数化技能： 这是计算的根基。

明确区分两类积分： 问自己：被积函数是标量还是向量？积分域有方向要求吗？

明确三大公式的使用条件： 封闭性？方向（正向/外侧/右手定则）？

多画图： 画出曲线/曲面、方向、向量场，建立空间直观感受。

从简单例子入手： 先计算直线段、平面区域上的积分，再过渡到曲线和曲面。

理解这些核心概念和联系，再结合具体的计算方法练习，就能逐步掌握曲线积分和曲面积分。记住，公式只是工具，理解背后的“为什么”才是关键。
