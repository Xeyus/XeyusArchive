# 学习与应用

### 1.学习是提炼信息的过程

1.1 信息包括感官获取的信号、语言、实践经历
1.1.1 所以学习的上限，理解的上限，基于我们的物质组成和结构
1.2 我们难以把所有习得的知识一一阐释，因为它们可能是理所当然的或是难以表述的
1.2.1 比如定量、定义这类必然成立的基础
1.2.2 再比如没有符号简化时的遗传法则和解方程
1.2.3 这导致了我们难以进一步学习知识，以及更重要的，教授
1.3 一切思想与意志建立在该过程之上。强调某一信息、某一意志的重要性而不讲述如何学习之，无异于一味用荧光笔勾画重点而不写解释
1.4 同一物品或概念的信息被不同人关注，因此为了学习，平等是必要的
1.5 组织习得信息的标准是：可拓展，可接受改变，有逻辑链条
1.5.1 让它成为一个有复杂结构的花园，而不是单线程的“XX 就是 XX，XX 让我 XX”。
1.5.2 不必强求分类。它们无法被准确定义，也难以抓住所谓的主要特点
1.6 信息不必来自专业。信息需要来自易被接受理解的地方
1.6.1 也是为什么垃圾信息流行
1.7 信息只是一段呈现了自身的信号，而非真理，不论其来自何方
1.7.1 运用事实 - 情感 - 逻辑交叉判断其真实性，如情感过于刻奇是否有问题
1.7.2 不要否定陈述。否定它的表现形式、内容、逻辑、情感

### 2.按照重要程度决定信息的归宿

2.1 自己的思考与总结必是最优
2.1.1 相关证明不必记忆，但需记述

### 3.信源需要广泛的选择

3.1 从一本书延伸
3.1.1 同一个作者
3.1.2 寻找你感兴趣的和它的中心主题
3.1.3 注释、引用、致谢
3.2 具体的人
3.2.1 领域先锋
3.2.2 领域中有经验的人
