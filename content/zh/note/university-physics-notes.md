#### 理想气体的四种过程

*等压、等体、等温、绝热过程*

**基本公式**

**理想气体状态方程：**
$pV = \nu RT = \frac{m}{M}RT$
其中 $\nu$ 为摩尔数，$R = 8.31$ J/(mol·K) 为气体常数

**内能：**
$E = \frac{i}{2}\nu RT$
其中 $i$ 为自由度（单原子 $i=3$，双原子 $i=5$，多原子 $i=6$）

**热力学第一定律：**
$Q = \Delta E + W$

**功：**
$W = \int p dV$

**摩尔热容定义：**
$C = \frac{1}{\nu}\frac{dQ}{dT}$

**重要概念**
- **自由度：** 决定分子能量分配的独立运动方式数目
- **能量均分定理：** 每个自由度平均能量为 $\frac{1}{2}kT$（或 $\frac{1}{2}RT$ /摩尔）
- **绝热线比等温线陡：** 在 $p-V$ 图上，绝热线斜率绝对值更大
- **热容关系：** $C_p > C_V$（等压过程既增加内能又对外做功）

> **💡 例题 1：等压过程**
> 
> **题目：** 1 mol 双原子理想气体在等压过程中温度升高 50 K，求：(1) 内能增量 (2) 对外做功 (3) 吸收热量
> 
> **解：**
> 
> 双原子气体：$C_V = \frac{5}{2}R$，$C_p = \frac{7}{2}R$
> 
> (1) $\Delta E = \nu C_V \Delta T = 1 \times \frac{5}{2} \times 8.31 \times 50 = 1039 \text{ J}$
> 
> (2) $W = \nu R\Delta T = 1 \times 8.31 \times 50 = 416 \text{ J}$
> 
> (3) $Q = \nu C_p \Delta T = 1 \times \frac{7}{2} \times 8.31 \times 50 = 1454 \text{ J}$
> 
> 验证：$Q = \Delta E + W = 1039 + 416 = 1455 \text{ J}$ ✓

> **💡 例题 2：绝热过程**
> 
> **题目：** 理想气体从状态 $(p_1, V_1, T_1)$ 绝热膨胀到体积 $V_2 = 2V_1$，已知 $\gamma = 1.4$，求末态压强 $p_2$ 和温度 $T_2$
> 
> **解：**
> 
> 使用绝热过程方程：
> 
> (1) $p_1V_1^\gamma = p_2V_2^\gamma$
> 
> $p_2 = p_1\left(\frac{V_1}{V_2}\right)^\gamma = p_1\left(\frac{1}{2}\right)^{1.4} = p_1 \times 0.379 = 0.379p_1$
> 
> (2) $T_1V_1^{\gamma-1} = T_2V_2^{\gamma-1}$
> 
> $T_2 = T_1\left(\frac{V_1}{V_2}\right)^{\gamma-1} = T_1\left(\frac{1}{2}\right)^{0.4} = T_1 \times 0.758 = 0.758T_1$

> **💡 例题 3：等温与绝热比较**
> 
> **题目：** 1 mol 理想气体分别经历等温膨胀和绝热膨胀，体积从 $V_1$ 增大到 $V_2 = 2V_1$，初态温度 300 K，$\gamma = 1.4$。求两种过程中气体对外做功的差值。
> 
> **解：**
> 
> **等温过程：**
> 
> $W_{等温} = \nu RT_1 \ln\frac{V_2}{V_1} = 1 \times 8.31 \times 300 \times \ln 2$
> 
> $W_{等温} = 2493 \times 0.693 = 1728 \text{ J}$
> 
> **绝热过程：**
> 
> $T_2 = T_1\left(\frac{V_1}{V_2}\right)^{\gamma-1} = 300 \times 0.758 = 227.4 \text{ K}$
> 
> $W_{绝热} = \nu C_V(T_1 - T_2) = 1 \times \frac{5}{2} \times 8.31 \times (300 - 227.4)$
> 
> $W_{绝热} = 20.775 \times 72.6 = 1508 \text{ J}$
> 
> **差值：** $\Delta W = W_{等温} - W_{绝热} = 1728 - 1508 = 220 \text{ J}$
> 
> 结论：等温膨胀做功更多（因为吸收热量维持温度）

#### 卡诺热机

*可逆循环过程，理想热机效率*

**卡诺循环效率：**
$$\eta = 1 - \frac{T_2}{T_1}$$
其中 $T_1$ 为高温热源温度，$T_2$ 为低温热源温度

**吸热与放热：**
$$\frac{Q_1}{Q_2} = \frac{T_1}{T_2}$$

**做功：**
$$W = Q_1 - Q_2 = Q_1\left(1 - \frac{T_2}{T_1}\right)$$

**熵变：**
$$\Delta S = 0$$
可逆循环总熵变为零

### 波动和振动

##### 简谐运动

**简谐振动方程：**
$x = A\cos(\omega t + \varphi)$
其中：
- $A$ 为振幅
- $\omega$ 为角频率
- $\varphi$ 为初相位
- $\omega t + \varphi$ 为相位
**速度和加速度：**
$v = \frac{dx}{dt} = -A\omega\sin(\omega t + \varphi)$
$a = \frac{dv}{dt} = -A\omega^2\cos(\omega t + \varphi) = -\omega^2 x$
**角频率、周期、频率关系：**
$\omega = \sqrt{\frac{k}{m}} = 2\pi f = \frac{2\pi}{T}$
**由初始条件确定振幅和初相：**
$A = \sqrt{x_0^2 + \frac{v_0^2}{\omega^2}}$
$\tan\varphi = -\frac{v_0}{\omega x_0}$

> 已知时刻/图像求简谐运动初相
> 例：一物体沿x轴作简谐振动，振幅为0.24m，当t=0时，在x=0.12m处，且向x轴正方向运动，请写出振动的初相。
> 解：$x = A\cos(\omega t + \varphi)$，则 $x_0 = A cos φ,0.12 = 0.24 cos φ,\cos φ = 0.5$，所以 $φ=\frac{\pi}{3}\text{或}-\frac{\pi}{3}$，又因为向正方向运动，$v = -Aω sin (ωt + φ)$，$t_{0}$ 时 $v_0 = -Aω sin φ$，所以需要 $\sin φ<0$，即 $φ=-\frac{\pi}{3}$
> 给出图像时同理。

> 求简谐运动方程
> 例：已知质点简谐振动曲线x~t，求质点的振动方程。
> 解：根据给出的点，确定 A，$\varphi$，进而根据相位关系求ω：$φ_{1} = ωt_{1} + φ_{0}$，$x_{1}=A\cos(\omega t_{1}+\varphi)$，代入得ω。

> 已知振动方程，求从某位置到另一位置的时间
> 例：已知一物体作简谐振动，振动方程为 $x=0.24cos\left( \pi t-\frac{\pi}{3} \right)$ ，试求从 x=-0.12 m，且向x轴负方向运动的状态回到平衡位置所需的最短时间。
> 解：代入得初始位置至所求位置的最短相位，$t=\frac{\varphi}{\omega}$
> 相位差的计算：
> **方法一：几何法（推荐）**
>  - 画出运动轨迹
>  - 在相位圆（或波形图）上标出起点和终点
>  - 数出最短圆心角
>  **方法二：代数法**
>  φ₁ = arcsin(x₁/A)  （注意象限）
> φ₂ = arcsin(x₂/A)  （注意象限）
> Δφ = φ₂ - φ₁

**简谐运动的**
**动能：**
$E_k = \frac{1}{2}mv^2 = \frac{1}{2}m\omega^2 A^2\sin^2(\omega t + \varphi)$
平均动能 $\overline{E_{k}}=\frac{1}{4}kA^2$
**势能：**
$E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\cos^2(\omega t + \varphi)$
平均动能 $\overline{E_{p}}=\frac{1}{4}kA^2$
**总能量（守恒）：**
$E = E_k + E_p = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2 A^2$
其中 $m\omega^2 = k$

**重要概念**
- **简谐振动特征：** 回复力 $F = -kx$，加速度 $a = -\omega^2 x$
- **能量特点：** 总能量与振幅平方成正比，动能和势能相互转化
- **相位：** 描述振动状态，相位差决定两振动的步调关系

**典型简谐振动系统**
1. 弹簧振子
	$\omega = \sqrt{\frac{k}{m}}, \quad T = 2\pi\sqrt{\frac{m}{k}}$，$A = \sqrt{x_0^2 + \frac{v_0^2}{\omega^2}}$
	求振子在某时刻的速度与加速度：$v=\frac{dx}{dt},a=\frac{dv}{dt}$，求导后代入即可。
2. 单摆
	**小角度近似下：**
	$\omega = \sqrt{\frac{g}{l}}, \quad T = 2\pi\sqrt{\frac{l}{g}}$
3. 复摆（物理摆）
	**运动方程：**
	$\frac{d^2\theta}{dt^2} = -\frac{mgl}{J}\theta$
	**角频率和周期：**
	$\omega = \sqrt{\frac{mgl}{J}}, \quad T = 2\pi\sqrt{\frac{J}{mgl}}$
	其中 $J$ 为对转轴的转动惯量，$l$ 为质心到转轴的距离

**判断简谐运动相位关系**
已知 $x_{1}=A_{1}\cos\omega t+\varphi_{1},x_{2}=A_{2}\cos \omega t+\varphi_{2}$，$\Delta \varphi=\varphi_{2}-\varphi_{1}$
- $\Delta \varphi=0$，$x_{1}$ 与 $x_{2}$ 同相
- $\Delta \varphi>0$，$x_{1}$ 滞后于 $x_{2}$
- $\Delta \varphi<0$，$x_{1}$ 超前于 $x_{2}$
- $\Delta \varphi=±\pi$，$x_{1}$ 与 $x_{2}$ 反相

**简谐运动的合成**
1. 同方向同频率简谐振动的合成
	**两个振动：**
	$x_1 = A_1\cos(\omega t + \varphi_1)$
	$x_2 = A_2\cos(\omega t + \varphi_2)$
	**合振动：**
	$x = A\cos(\omega t + \varphi)$
	**合振幅：**
	$A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos(\varphi_2 - \varphi_1)}$
	**合振动初相：**
	$\tan\varphi = \frac{A_1\sin\varphi_1 + A_2\sin\varphi_2}{A_1\cos\varphi_1 + A_2\cos\varphi_2}$
	**特殊情况：**
	- 相位差 $\Delta\varphi = 2k\pi$（同相）：$A = A_1 + A_2$（加强）
	- 相位差 $\Delta\varphi = (2k+1)\pi$（反相）：$A = |A_1 - A_2|$（减弱）
	- 一般情况：$|A_1 - A_2| \leq A \leq A_1 + A_2$
2. 相互垂直的同频率简谐振动的合成
	**两个振动：**
	$x = A_1\cos(\omega t + \varphi_1)$
	$y = A_2\cos(\omega t + \varphi_2)$
	**轨迹方程（消去 $t$）：**
	$\frac{x^2}{A_1^2} + \frac{y^2}{A_2^2} - \frac{2xy}{A_1A_2}\cos(\varphi_2 - \varphi_1) = \sin^2(\varphi_2 - \varphi_1)$
	**特殊情况：**
	- $\Delta\varphi = 0$：$y = \frac{A_2}{A_1}x$（直线，斜率为正）
	- $\Delta\varphi = \pi$：$y = -\frac{A_2}{A_1}x$（直线，斜率为负）
	- $\Delta\varphi = \pm\frac{\pi}{2}$：$\frac{x^2}{A_1^2} + \frac{y^2}{A_2^2} = 1$（椭圆）
	  - $\Delta\varphi = \frac{\pi}{2}$：逆时针椭圆（左旋）
	  - $\Delta\varphi = -\frac{\pi}{2}$：顺时针椭圆（右旋）
	  - 若 $A_1 = A_2$，则为圆
3. 同方向不同频率简谐振动的合成（拍）
	**两个振动：**
	$x_1 = A\cos(\omega_1 t)$
	$x_2 = A\cos(\omega_2 t)$
	设 $A_1 = A_2 = A$，$\varphi_1 = \varphi_2 = 0$，$\omega_1 \approx \omega_2$
	**合振动：**
	$x = x_1 + x_2 = 2A\cos\left(\frac{\omega_1 - \omega_2}{2}t\right)\cos\left(\frac{\omega_1 + \omega_2}{2}t\right)$
	可写成：
	$x = A'\cos(\bar{\omega}t)$
	其中：
	- 振幅：$A' = 2A\left|\cos\left(\frac{\omega_1 - \omega_2}{2}t\right)\right|$（缓变）
	- 平均角频率：$\bar{\omega} = \frac{\omega_1 + \omega_2}{2}$（快变）
	**拍的特征：**
	- **拍频：** $f_{拍} = |f_1 - f_2| = |\nu_1 - \nu_2|=\frac{\omega_{1}}{2\pi}-\frac{\omega_{2}}{2\pi}$
	- **拍周期：** $T_{拍} = \frac{1}{|f_1 - f_2|}$
	- **物理意义：** 合振动的振幅周期性增强和减弱

##### 机械波

**平面简谐波的波函数：**

**以 $x_1$ 处质点为参考：**
$y(x, t) = A\cos\left[\omega\left(t - \frac{x - x_1}{u}\right) + \varphi\right]$
或写成：
$y(x, t) = A\cos\left[\omega t - k(x - x_1) + \varphi\right]$

**以原点为参考（$x_1 = 0$）：**
$y(x, t) = A\cos(\omega t - kx + \varphi)$
其中：
- $A$ 为振幅
- $\omega$ 为角频率
- $k = \frac{2\pi}{\lambda} = \frac{\omega}{u}$ 为波数
- $u$ 为波速
- $\varphi$ 为初相位

**波函数的其他形式：**
$y(x, t) = A\cos\left[2\pi\left(\frac{t}{T} - \frac{x}{\lambda}\right) + \varphi\right]$
$y(x, t) = A\cos\left[2\pi\left(ft - \frac{x}{\lambda}\right) + \varphi\right]$

**波速、频率、波长关系：**
$u = \lambda f = \frac{\lambda}{T}$

**相位：**
$\phi(x, t) = \omega t - kx + \varphi$

**波的传播方向：**
- 沿 $+x$ 方向：$y = A\cos(\omega t - kx + \varphi)$
- 沿 $-x$ 方向：$y = A\cos(\omega t + kx + \varphi)$

> 已知波函数，求某刻/某点振动方程/速度：直接代入/求导后代入。
> 文字/图像描述波动，求波动方程：根据上述式子 $y(x, t) = A\cos\left[2\pi\left(\frac{t}{T} - \frac{x}{\lambda}\right) + \varphi\right]$ 计算，其中 $\lambda=uT,T=\frac{1}{f}=\frac{2\pi}{\omega}$，及波的传播方向判定得方程。
> 求初相 $\varphi$：y 对 t 求导得到 v，将已知的 t、x 代入 v 表达式，根据 v 正负判断取哪个 $\varphi$。
> 两质元相距 $\Delta x=\frac{\Delta \varphi}{2\pi}\lambda$

**重要概念**

- **波线（波射线）：** 箭头指向波的传播方向
- **波阵面（波前）：** 同一时刻相位相同的点连成的曲面
  - 平面波：波阵面为平面
  - 球面波：波阵面为球面
- **波长 $\lambda$：** 沿传播方向，相邻两个相位差为 $2\pi$ 的点之间的距离
- **周期 $T$：** 波传播一个波长的时间

**判断振动方向的方法：**
- "下坡下，上坡上"：沿波传播方向看，质点在"下坡"处向下运动，"上坡"处向上运动
- 或用 $v_y = \frac{\partial y}{\partial t}$ 计算
##### 波的能量

**能量密度：**
$w = \frac{1}{2}\rho\omega^2 A^2\sin^2(\omega t - kx + \varphi)$

**平均能量密度：**
$\bar{w} = \frac{1}{4}\rho\omega^2 A^2$

**能流密度（波的强度）：**
$I = \bar{w} \cdot u = \frac{1}{4}\rho u\omega^2 A^2$

##### 波的干涉

*相干波叠加产生加强和减弱*

**相干条件**

1. 频率相同
2. 振动方向相同
3. 相位差恒定

**波的叠加原理**：介质中任一点的位移等于各波单独传播时在该点产生位移的矢量和。

**干涉条件**

**两相干波源 $S_1$ 和 $S_2$ 在点 $P$ 处的相位差：**
$\Delta\varphi = (\varphi_2 - \varphi_1) - k(r_1 - r_2)$
$= (\varphi_2 - \varphi_1) - \frac{2\pi}{\lambda}(r_1 - r_2)$
其中 $r_1$、$r_2$ 为两波源到 $P$ 点的距离，$\varphi_1$、$\varphi_2$ 为两波源的初相位。

**干涉加强（振幅最大）：**
$\Delta\varphi = 2k\pi \quad \text{或} \quad \delta = r_1 - r_2 = k\lambda, \quad k = 0, \pm1, \pm2, \ldots$

**干涉减弱（振幅最小）：**
$\Delta\varphi = (2k+1)\pi \quad \text{或} \quad \delta = r_1 - r_2 = (2k+1)\frac{\lambda}{2}, \quad k = 0, \pm1, \pm2, \ldots$

**波程差：** $\delta = r_1 - r_2$（光程差的概念）

##### 驻波

*两列振幅相同、频率相同、传播方向相反的波叠加形成*

**驻波方程**：$y = 2A\cos(kx)\sin(\omega t)$

**驻波特征：**

**波节：** $\cos(kx) = 0$
$x = (2n+1)\frac{\lambda}{4}, \quad n = 0, 1, 2, \ldots$

**波腹：** $|\cos(kx)| = 1$
$x = n\frac{\lambda}{2}, \quad n = 0, 1, 2, \ldots$

**相邻波节（或波腹）间距：** $\frac{\lambda}{2}$

**驻波特点：**
- 各点振幅不同（振幅为 $2A|\cos(kx)|$）
- 波节处振幅为零，波腹处振幅最大
- 位于相邻两个波节之间的所有质点，它们的振动相位完全相同；位于任一波节两侧的质点，其振动相位正好相反

##### 多普勒效应

*波源或观察者运动时，观察到的频率发生变化*

**基本公式**： $f' = \frac{u \pm v_观}{u \mp v_源}f$

**符号规定：**
以**观察者指向声源**为正方向
- 观察者靠近波源：$v_观$ 取正号
- 观察者远离波源：$v_观$ 取负号  
- 波源靠近观察者：$v_源$ 取正号（分母取负号）
- 波源远离观察者：$v_源$ 取负号（分母取正号）

> 例：一警车以v1=25m/s在静止的空气中追赶v2=15m/s的客车，声速u=330m/s，警车警笛频率800Hz，则客车上人听到警笛频率是多少？
> 在本题中：**客车（观察者）→ 警车（声源）方向为正**
> - 客车向右走，这是**背离"客车→警车"方向**：v_观 = -15 m/s
> - 警车向右走，这是**沿着"客车→警车"方向**：v_源 = +25 m/s
> - 所以 $f' = \frac{u - v_观}{u - v_源}f$

**特殊情况：**
- 观察者静止，波源运动：$f' = \frac{u}{u \mp v_源}f$
- 波源静止，观察者运动：$f' = \frac{u \pm v_观}{u}f$

#### 相对论中的动量和能量*高速运动下的能量动量关系*

**基本公式**

**相对论能量：**
$$E = \gamma mc^2 = \frac{mc^2}{\sqrt{1-v^2/c^2}}$$

**静止能量：**
$$E_0 = mc^2$$

**动能：**
$$E_k = (\gamma - 1)mc^2 = E - E_0$$

**相对论动量：**
$$p = \gamma mv = \frac{mv}{\sqrt{1-v^2/c^2}}$$

**能量动量关系（重要！）：**
$$E^2 = (pc)^2 + (mc^2)^2$$

**重要概念**
- **洛伦兹因子：** $\gamma = \frac{1}{\sqrt{1-v^2/c^2}}$
- **低速近似：** $E_k \approx \frac{1}{2}mv^2$（经典动能）
- **光子情况：** $E = pc$（$m=0$）

> **💡 例题**
> 
> **题目：** 电子以 $0.6c$ 的速度运动，求其动能与静止能量的比值（电子静止能量为 0.511 MeV）
> 
> **解：**
> 
> $\gamma = \frac{1}{\sqrt{1-0.6^2}} = \frac{1}{\sqrt{0.64}} = 1.25$
> 
> $E_k = (\gamma - 1)mc^2 = 0.25 \times 0.511 = 0.128 \text{ MeV}$
> 
> $\frac{E_k}{E_0} = 0.25 = 25\%$

**德布罗意波长：**
$$\lambda = \frac{h}{p} = \frac{h}{mv}$$

**位置-动量不确定性关系：**
$$\Delta x \Delta p \geq \frac{\hbar}{2}$$
其中 $\hbar = \frac{h}{2\pi}$

**能量-时间不确定性关系：**
$$\Delta E \Delta t \geq \frac{\hbar}{2}$$

**平面波波函数：**
$$\psi(x,t) = A e^{i(kx - \omega t)}$$
其中 $k = \frac{2\pi}{\lambda}$，$\omega = 2\pi\nu$

**重要概念**
- **约化普朗克常数：** $\hbar = 1.055 \times 10^{-34}$ J·s
- **波粒二象性：** 微观粒子既有波动性又有粒子性
- **经典极限：** 宏观物体波长极小，波动性不明显

> **💡 例题**
> 
> **题目：** 电子经 100 V 电压加速后的德布罗意波长是多少？（$m_e = 9.11 \times 10^{-31}$ kg）
> 
> **解：**
> 
> $E_k = eU = 100 \text{ eV} = 1.6 \times 10^{-17} \text{ J}$
> 
> $p = \sqrt{2m_e E_k} = \sqrt{2 \times 9.11 \times 10^{-31} \times 1.6 \times 10^{-17}}$
> 
> $p = 5.4 \times 10^{-24} \text{ kg·m/s}$
> 
> $\lambda = \frac{h}{p} = \frac{6.63 \times 10^{-34}}{5.4 \times 10^{-24}} = 1.23 \times 10^{-10} \text{ m}$

#### 波函数

*量子态描述，概率诠释*

**基本公式**

**波函数归一化：**
$$\int_{-\infty}^{+\infty} |\psi|^2 dx = 1$$

**概率密度：**
$$\rho(x) = |\psi(x)|^2$$

**一维无限深势阱波函数：**
$$\psi_n = \sqrt{\frac{2}{L}} \sin\left(\frac{n\pi x}{L}\right), \quad n = 1, 2, 3, \ldots$$

**能级公式：**
$$E_n = \frac{n^2 h^2}{8mL^2}$$

**重要概念**
- **概率诠释：** $|\psi|^2$ 表示粒子出现的概率密度
- **波函数条件：** 必须连续、单值、有限
- **能量量子化：** $E_{n+1} - E_n = \frac{(2n+1)h^2}{8mL^2}$

> **💡 例题**
> 
> **题目：** 无限深势阱（宽度 $L$）中，粒子处于基态（$n=1$），求在 $\frac{L}{4}$ 到 $\frac{3L}{4}$ 区间找到粒子的概率
> 
> **解：**
> 
> $\psi_1 = \sqrt{\frac{2}{L}} \sin\left(\frac{\pi x}{L}\right)$
> 
> $P = \int_{L/4}^{3L/4} |\psi_1|^2 dx = \frac{2}{L} \int_{L/4}^{3L/4} \sin^2\left(\frac{\pi x}{L}\right) dx$
> 
> 计算得：$P \approx 0.818$（约 82%）

#### 氢原子玻尔理论

*定态、跃迁、光谱*

**基本公式**

**角动量量子化：**
$$L = n\hbar, \quad n = 1, 2, 3, \ldots$$

**轨道半径：**
$$r_n = n^2 a_0$$
其中 $a_0 = 0.0529$ nm 为玻尔半径

**能级公式：**
$$E_n = -\frac{13.6}{n^2} \text{ eV}$$

**跃迁条件：**
$$h\nu = E_n - E_m$$

**里德伯公式：**
$$\frac{1}{\lambda} = R\left(\frac{1}{m^2} - \frac{1}{n^2}\right)$$
其中 $R = 1.097 \times 10^7$ m $^{-1}$ 为里德伯常数

**重要概念**
- **基态能量：** $E_1 = -13.6$ eV
- **电离能：** 13.6 eV
- **光谱系：** 
  - 莱曼系（$m=1$）：紫外
  - 巴尔末系（$m=2$）：可见光

> **💡 例题**
> 
> **题目：** 氢原子从 $n=3$ 跃迁到 $n=2$，求发射光子的波长和频率
> 
> **解：**
> 
> $E_3 = -\frac{13.6}{9} = -1.51 \text{ eV}$
> 
> $E_2 = -\frac{13.6}{4} = -3.40 \text{ eV}$
> 
> $\Delta E = E_3 - E_2 = 1.89 \text{ eV} = 3.02 \times 10^{-19} \text{ J}$
> 
> $\lambda = \frac{hc}{\Delta E} = 656 \text{ nm}$（红色，巴尔末系 H $_\alpha$ 线）

#### 氢原子量子理论

*薛定谔方程，量子数*

**基本公式**

**能量：**
$$E_n = -\frac{13.6}{n^2} \text{ eV}$$
$n = 1, 2, 3, \ldots$ 为主量子数

**角动量大小：**
$$L = \sqrt{l(l+1)} \hbar$$
$l = 0, 1, \ldots, n-1$ 为角量子数

**角动量 $z$ 分量：**
$$L_z = m_l \hbar$$
$m_l = -l, \ldots, 0, \ldots, +l$ 为磁量子数

**自旋角动量：**
$$S = \sqrt{s(s+1)} \hbar$$
$s = \frac{1}{2}$，$S_z = \pm \frac{\hbar}{2}$

**重要概念**
- **四个量子数：** $n$（主），$l$（角），$m_l$（磁），$m_s$（自旋）
	- 对 ml，有 2 l-1 种，对 ms，有 2 种，对 l，有 n-1 种。所以固定 n, l，有 2 (2 l+1) 种不同的量子态，只固定 n，就是从 0 到 n-1 的 l 对应的加起来。
- **轨道符号：** $l = 0, 1, 2, 3$ 对应 $s, p, d, f$ 轨道
- **简并度：** 每个主量子数 $n$ 有 $n^2$ 个态（不计自旋）
- **精细结构：** 来自自旋-轨道耦合

> **💡 例题**
> 
> **题目：** $n=3$ 的氢原子能级有哪些可能的 $l$ 和 $m_l$ 值？共有多少个态（不计自旋）？
> 
> **解：**
> 
> $l$ 可取：$0, 1, 2$（3 s, 3 p, 3 d 轨道）
> 
> - $l=0$：$m_l = 0$（1 个态）
> - $l=1$：$m_l = -1, 0, +1$（3 个态）
> - $l=2$：$m_l = -2, -1, 0, +1, +2$（5 个态）
> 
> 总共：$1 + 3 + 5 = 9$ 个态
