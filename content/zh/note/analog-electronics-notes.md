## 半导体、二极管、三极管

### 半导体

- PN结：扩散（自发浓度高-低）和漂移（外加电场导致）；PN半导体结合形成的，由N指向P的具有阻止载流子扩散运动的内电场区，具有单向导电性。加正向电压（P正N负）时耗尽区变窄表现为小电阻，加反向电压时表现为类似断路。会反向击穿。
  - 电容效应：结电容=势垒电容+扩散电容
  - 外加正向电压时扩散电流大于漂移电流
  - 加正向一定导通，加反向不一定截止
- 本征半导体：纯净，导电能力弱
- 掺入离子：
  - +5（如P）: N型，自由电子>空穴，自由电子为多数载流子
  - +3（如B）: P型，自由电子<空穴，空穴为多数载流子
  - 但整体电平衡。本征激发（共价键分裂）产生电子空穴对

### 二极管

- 组成：PN结
- 类型
  - 硅管：开启电压0.5V，线性区0.6-1V，反向电流0.1mA，正向压降$U_{D}=0.7V$
  - 锗管：开启电压0.1V，线性区0.2-0.5V，反向电流几十mA，正向压降$U_{D}=0.2V$
- 符号
- 伏安特性
  - 正向
    - 开启电压
    - 反向饱和电流
  - 反向
    - $U_{BR}$，几十V以上
    - 反向饱和电流$I_{s}$
  - 通用
    - 受温度影响：温度升高，正向特性左移（$I不变，U下移$），反向特性下移（$I_{s}增大，U_{BR}减小$）
- 主要参数
- 等效电路
  - 理想二极管
  - 带正向压降的元件
- 计算方法
  - 判断工作状态：断开，查看两侧压降，正偏则N-P>0，否则反偏断开。多个则压差大的先导通
  - 根据工作状态换成等效电路。理想状态下当开关，非理想情况按照$U_{D}$情况等效
  - 列电路方程
  - 稳压二极管：工作条件有最大电流要求，需要反向偏压进入工作状态（稳压）。同理并联时谁稳定电压值更小，也就是离反偏电压更远，谁先被击穿然后维持在该稳压。

- 题型
  - 判断导通还是截止
  - 含二极管的电路分析：电压、电流平均值，正弦输入经二极管后的输出函数/图像

#### 三极管

- 三极管
  - 组成
    - c：集电极
    - b：基极
    - e：发射极
    - 掺杂浓度e>>c>>b
  - 类型与符号
    - PNP箭头向基极
    - NPN箭头向发射极
  - 工作：
    - 截止区：均反偏
    - 工作区：发射结正偏，集电结反偏
    - 饱和区：均正偏
  - 题型
    - 静态（直流）电路分析
    - 交流电路分析
    - 根据特性曲线或工作数据判断参数

##### 判断共射/共基：

- 画出交流通路：将电容短路，将直流电源短路
- 定位输入、负载电流流入的电极
- 判断公共端
  - E交流接地/接电源，输入B输出C，共基极电路CB
  - C接电源，输入B，输出E，共集电极电路 CC
  - B为公共参考点，输入B，输出C，共发射极电路 CE
  - 即- 哪个电极**交流接地**（或接电源），就是**共XX**电路
    - 发射极接地 → 共射
    - 集电极接地 → 共集
    - 基极接地 → 共基

#### 场效应管

- 场效应管
  - 组成
    - g（gate栅极）-对应b
    - d（drain漏极）-对应c
    - s（source源极）-对应e
  - 类型与符号
    - MOSFET，N沟道
      - 耗尽型
      - 增强型
    - MOSFET，P沟道
      - 耗尽型
      - 增强型
    - JFET，N沟道
    - JFET，P沟道
    - 其中重点区分JFET和MOSFET（增强型/耗尽型）。
    - 图像特点：
      - **MOSFET**：栅极（G）与沟道之间是绝缘的，符号中栅极线**不直接接触**沟道线，有一段间隙。
      - **JFET**：栅极与沟道直接形成 PN 结，符号中栅极线**直接接触**沟道线（无间隙）。
      - 箭头**指向沟道**（向内）→ **N 沟道**。
      - 箭头**远离沟道**（向外）→ **P 沟道**。
      - 沟道线为**断开的虚线（或三段短线）** → **增强型**（默认无沟道）
      - 沟道线为**连续实线** → **耗尽型**（默认有沟道）。
      - JFET 符号中沟道线总是**连续实线**（因为默认存在沟道），无需区分增强/耗尽。
  - 主要参数
    - 共栅关系
    - 共源关系
    - 共漏关系
- 题型
  - 同晶体管放大电路。

### 晶体管

基本静态电路方程：
$V_{CC} = I_{C}·R_{C} + U_{CE}$
$V_{CC}= I_{B}·R_{B} + U_{BE}$
$I_{C} = β·I_{B}$

Rc对Q影响：$i_{c}=-{\frac{1}{R_{c}}u_{CE}+\frac{V_{C C}}{R_{c}}}$有Rc增加Q左移，同理$i_{b}=-{\frac{1}{R_{b}}u_{BE}+\frac{V_{B B}}{R_{b}}}$有Rb增加Q下移

Q一般让写（$I_{CQ},U_{CEQ}$），但是B上电流也是静态工作点参数。

#### 静态工作点

![[{{notename}}-2026-03-10-152952-1.webp]]

1. 情况一：无电阻并在b极与地之间
   1. 将b极与$V_{ CC}$之间的电阻记作$R_{b}$，c极上的电阻记作$R_{c}$，e极上的电阻记作$R_{e}$
   2. $I_{BQ}=\frac{V_{CC} - U_{BEQ}}{(1+\beta)R_{e}+R_{b}}$，$I_{CQ}=\beta I_{BQ}$，$U_{CEQ}=V_{C C}-I_{CQ}R_{c}-(1+\beta)I_{BQ}R_{e}$
   3. 其中$U_{BEQ}$题干中会给，默认硅管0.7V锗管0.3V
2. 情况二：有电阻并在b极与地之间，且e极上有电阻
   1. 将b极与$V_{ CC}$之间的电阻记作$R_{b1}$，b极与地之间的电阻记作$R_{b2}$，c极上的电阻记作$R_{c}$，e极上的电阻记作$R_{e}$
   2. $I_{BQ}=\frac{1}{(1+\beta)R_{e}}\left( \frac{R_{b 2}}{R_{b 1}+R_{b 2}}V_{CC}-U_{BEQ} \right)$，$I_{CQ}=\beta I_{BQ}$，$U_{CEQ}=V_{C C}-I_{CQ}R_{c}-(1+\beta)I_{BQ}R_{e}$
   3. 即$I_{BQ}$考虑b和地之间电阻分压（$V_{C C}的系数$）
      - 基极电压由分压器决定：$U_B = \frac{R_{b2}}{R_{b1}+R_{b2}}V_{CC}$
      - 从基极-发射极回路看：$U_B = U_{BEQ} + I_ER_e$
      - 发射极电流 $I_E = (1+\beta)I_B$
3. 情况三：有电阻并在b极与地之间，且e极上无电阻
   1. 将b极与$V_{ CC}$之间的电阻记作$R_{b1}$，b极与地之间的电阻记作$R_{b2}$，c极上的电阻记作$R_{c}$
   2. $I_{BQ}=\frac{V_{CC} - U_{BEQ}}{R_{b 1}}-\frac{U_{BEQ}}{R_{b 2}}$，$I_{CQ}=\beta I_{BQ}$，$U_{CEQ}=V_{C C}-I_{CQ}R_{c}$
   3. 因为e上没有电阻，所以
      - 由于发射极直接接地，$U_E = 0$，所以 $U_B = U_{BEQ}$
      - 基极节点电流：从 $V_{CC}$ 通过 $R_{b1}$ 的电流为 $I_1 = \frac{V_{CC}-U_B}{R_{b1}}$
      - 通过 $R_{b2}$ 的电流为 $I_2 = \frac{U_B}{R_{b2}}$
      - 根据KCL：$I_{BQ} = I_1 - I_2 = \frac{V_{CC}-U_B}{R_{b1}} - \frac{U_B}{R_{b2}}$
      - 代入 $U_B = U_{BEQ}$ 即得公式

### 场效应管

#### 静态工作点

其直流等效电路绘制方法同晶体管，不重复叙述。

1. 情况一：自偏压电路（适用于JFET和耗尽型MOSFET）
   1.1 电路描述
   - 栅极通过电阻 $R_g$ 接地（或直接接地）
   - 源极有电阻 $R_s$，漏极有电阻 $R_d$
     1.2 静态工作点公式
   - $U_{GSQ} = -I_{DQ}R_s$
   - $I_{DQ} = I_{DSS}\left(1-\frac{U_{GSQ}}{U_P}\right)^2$（JFET和耗尽型MOSFET）
   - $U_{DSQ} = V_{DD} - I_{DQ}(R_d + R_s)$
     1.3 公式解释
   - 栅极电流为零，栅极电压 $U_G = 0$
   - 源极电压 $U_S = I_{DQ}R_s$
   - $U_{GSQ} = U_G - U_S = -I_{DQ}R_s$
   - 将 $U_{GSQ}$ 代入FET电流方程求解 $I_{DQ}$
2. 情况二：分压器偏置电路（适用于所有FET类型）
   2.1 电路描述
   - 栅极通过两个电阻 $R_{g1}$ 和 $R_{g2}$ 分压连接到 $V_{DD}$ 和地
   - 源极有电阻 $R_s$，漏极有电阻 $R_d$
     2.2 静态工作点公式
   - $U_G = \frac{R_{g2}}{R_{g1}+R_{g2}}V_{DD}$
   - $U_S = I_{DQ}R_s$
   - $U_{GSQ} = U_G - U_S = \frac{R_{g2}}{R_{g1}+R_{g2}}V_{DD} - I_{DQ}R_s$
   - JFET/耗尽型MOSFET：$I_{DQ} = I_{DSS}\left(1-\frac{U_{GSQ}}{U_P}\right)^2$
   - 增强型MOSFET：$I_{DQ} = K(U_{GSQ}-U_T)^2$
   - $U_{DSQ} = V_{DD} - I_{DQ}(R_d + R_s)$
     2.3 公式解释
   - 栅极电压 $U_G$ 由分压器固定
   - 源极电压 $U_S$ 由 $I_{DQ}$ 和 $R_s$ 决定
   - $U_{GSQ}$ 是两者之差，代入FET电流方程求解

###### 根据输入输出图像画出/求解静态工作点

   2.1 基本图解原理
      2.1.1 转移特性分析
         - **转移特性曲线**：$I_D$ vs $U_{GS}$ 关系曲线
         - **偏置线**：根据偏置电路方程画出直线
         - **交点**：转移特性曲线与偏置线的交点即为静态工作点 $Q(I_{DQ}, U_{GSQ})$
      2.1.2 输出特性分析
         - **输出特性曲线**：$I_D$ vs $U_{DS}$ 关系曲线族（不同 $U_{GS}$ 对应不同曲线）
         - **直流负载线**：根据输出回路直流方程画出直线
         - **交点**：输出特性曲线与直流负载线的交点即为静态工作点 $Q(I_{DQ}, U_{DSQ})$
   2.2 情况一：自偏压电路图解
         2.2.2.1 转移特性分析：
            - 偏置线方程：$U_{GS} = -I_D R_s$
            - 在转移特性图上连接原点 $(0,0)$ 和点 $(-V_{DD}/R_s, V_{DD}/R_s)$ 画直线
            - 与转移特性曲线的交点得到 $I_{DQ}$ 和 $U_{GSQ}$
         2.2.2.2 输出特性分析：
            - 直流负载线方程：$U_{DS} = V_{DD} - I_D(R_d + R_s)$
            - 在输出特性图上连接点 $(0, V_{DD}/(R_d+R_s))$ 和 $(V_{DD}, 0)$ 画直线
            - 找到 $U_{GS} = U_{GSQ}$ 的特性曲线
            - 该曲线与直流负载线的交点得到 $I_{DQ}$ 和 $U_{DSQ}$
   2.3 情况二：分压器偏置电路图解
         2.3.2.1 转移特性分析：
            - 栅极电压：$U_G = \frac{R_{g2}}{R_{g1}+R_{g2}}V_{DD}$
            - 偏置线方程：$U_{GS} = U_G - I_D R_s$
            - 在转移特性图上连接点 $(0, U_G)$ 和点 $(U_G/R_s, 0)$ 画直线
            - 与转移特性曲线的交点得到 $I_{DQ}$ 和 $U_{GSQ}$
         2.3.2.2 输出特性分析：
            - 直流负载线方程：$U_{DS} = V_{DD} - I_D(R_d + R_s)$
            - 在输出特性图上连接点 $(0, V_{DD}/(R_d+R_s))$ 和 $(V_{DD}, 0)$ 画直线
            - 找到 $U_{GS} = U_{GSQ}$ 的特性曲线
            - 该曲线与直流负载线的交点得到 $I_{DQ}$ 和 $U_{DSQ}$
   2.4 通用图解技巧
      2.4.1 负载线绘制要点
         - 确定两个特殊点：横轴截距（$I=0$ 时的电压）和纵轴截距（$U=0$ 时的电流）
         - 用直尺连接两个截距点形成负载线
         - 确保负载线斜率正确：$-\frac{1}{R}$（其中 $R$ 为总负载电阻）
      2.4.2 交点确定方法
         - 先通过转移特性或输入特性确定控制量（$I_B$ 或 $U_{GS}$）
         - 再在输出特性上找到对应曲线
         - 该曲线与直流负载线的交点即为静态工作点
      2.4.3 验证方法
         - 检查工作点是否在放大区
         - 验证计算值与图解值的一致性
         - 确保工作点远离饱和区和截止区

##### 共源

电压放大倍数 $A_u=\frac{U_{o}}{U_{i}}$

- **无负载时**：$A_u = -g_m R_d$
- **有负载 $R_L$ 时**：$A_u = -g_m (R_d \parallel R_L)$
  源电压放大倍数 $A_{us}$
- $A_{us} = -\dfrac{g_m (R_d \parallel R_L)}{1 + g_m R_s}=\frac{R_{i}}{R_{i}+R_{s}}A_{u}=\frac{U_{o}}{U_{s}}$
  输入电阻：$R_i = R_G$
  输出电阻：$R_o = R_D$
  跨导：$g_m = 2 \sqrt{K_n I_{DQ}}$（增强型MOS管）  ，一般为已知项

##### 共漏

电压放大倍数 $A_u$

- **无负载时**：$A_u = \dfrac{g_m R_s}{1 + g_m R_s}$
- **有负载 $R_L$ 时**：$A_u = \dfrac{g_m (R_s \parallel R_L)}{1 + g_m (R_s \parallel R_L)}$
  源电压放大倍数
- $A_{vs} = \frac{v_o}{v_s} = A_v \cdot \frac{R_i}{R_i + R_s} = \frac{g_m (R_S \parallel R_L)}{1 + g_m (R_S \parallel R_L)} \cdot \frac{R_G}{R_G + R_s} \approx \frac{R_G}{R_G + R_s}$
  输入电阻：$R_i = R_G$
  输出电阻：$R_o = \frac{1}{g_m} \parallel R_S$
  跨导：$g_m = 2 \sqrt{K_n I_{DQ}}$（增强型MOS管） ，一般为已知项

注：参数说明

- $g_m$：跨导，$I_{DQ}$：静态漏极电流，$K_n$：MOS管增益因子
- $R_G、R_D、R_S$：栅极、漏极、源极电阻，$R_L$：负载电阻，$R_s$：信号源内阻
- $v_o$：输出电压，$v_i$：输入电压，$v_s$：信号源电压

##### 共栅

和共基一样，不用管。

### 放大电路的通频带

#### 波形失真

放大后输出信号与输入信号波形形状不一致的现象

##### 一、非线性失真（由器件非线性特性导致）

1. 截止失真
   - 条件：静态工作点设置偏低，输入信号 $u_i$ 幅值相对较大
   - 结果：$i_B、i_C、u_{CE}$ 波形出现顶部削平失真（NPN管共射电路中输出电压 $u_{CE}$ 削顶）
   - 消除方法：提高Q点位置（如减小基极电阻 $R_b$），或减小 $u_i$ 幅值
2. 饱和失真
   - 条件：静态工作点设置偏高，输入信号幅值相对较大
   - 结果：$i_C、u_{CE}$ 波形出现底部削平失真（NPN管共射电路中输出电压 $u_{CE}$ 削底）
   - 特点：由输出回路饱和状态（$u_{CE} \approx U_{CES}$，$U_{CES}$ 为饱和压降）引发
   - 消除方法：降低Q点位置（如增大 $R_b$ 或减小集电极电阻 $R_c$），或减小 $u_i$ 幅值
3. 交越失真
   - 条件：乙类互补对称功率放大电路中，晶体管无静态偏置($I_{BQ}=0、I_{CQ}=0$)，输入信号幅值较小时无法克服发射结死区电压
   - 结果：输出信号在正、负半周交界处出现失真（波形衔接不连续）
   - 特点：仅存在于推挽式功率放大电路，属于双管交替工作时的失真
   - 消除方法：采用甲乙类互补对称电路，给晶体管设置微小静态偏置（如通过二极管或电阻提供基极偏压），消除死区影响

对PNP管而言截止、饱和截止当然有相反性质。

##### 二、线性失真（由电路频率特性或相位特性导致，非器件非线性引起）

1. 频率失真
   - 分类：
     - 幅度失真：不同频率成分的信号放大倍数不同导致波形畸变
     - 相位失真：不同频率成分的信号相位偏移不同导致波形畸变
   - 条件：输入信号包含多种频率成分，且电路通频带无法覆盖所有频率
   - 特点：失真程度与信号频率相关，属于线性电路固有的频率响应问题
   - 改善方法：扩展电路通频带（如采用共射-共基组合电路）

##### 补充说明

- 非线性失真会产生新的频率成分（谐波），线性失真仅改变原有频率成分的幅度和相位
- 静态工作点是影响晶体管放大电路非线性失真的核心因素，合理设置Q点可减少截止/饱和失真
- 功率放大电路需重点关注交越失真，小信号放大电路更需避免截止/饱和失真
- 最大输出电压幅值：$U_{R}=U_{C EQ}-U_{CES}$，负载线方程：$V_{C C}=icRc +u_{CE},U_{F}=I_{CQ}Rc$，这些是重要计算方程。
  截止饱和两种失真均不发生的条件下，$U_{cemax}=min\{{U_{R},U_{F}}\}$，电路的最大输出电压幅值为$U_{omax}=min\{U_{CEQ}-U_{CES},I_{CQ}R_{C}\}$
  输出动态范围$U_{p-p}$表示最大输出电压的峰-峰值，所以$U_{p-p}=2U_{omax}$
  在电路条件一定的情况下，静态工作点设置使得$U_{R}=U_{F}$，这时放大电路有最大的输出电压幅值。

##### 通频带

- 通频带=上限截止频率－下限截止频率，$BW = f_H - f_L$
- 通频带 (Bandwidth, BW)：指放大电路的电压放大倍数幅值下降到其中频值的 **0.707倍** (即 **-3dB**) 时所对应的两个频率点之间的频率范围。
- $f_H$：**上限截止频率**。在高频段，由于晶体管极间电容和线路分布电容的影响，放大倍数开始下降。
- $f_L$：**下限截止频率**。在低频段，由于耦合电容和旁路电容的影响，放大倍数开始下降。
- **近似处理**：在许多情况下，$f_H >> f_L$，因此可以近似认为 $BW ≈ f_H$。

频率等于$f_H$时，电压放大倍数的变化

1. 核心结论：当频率等于上限截止频率 $f_H$ 时，电压放大倍数的幅值 $|A_u|$ 下降到其中频段幅值 $|A_m|$ 的 $\frac{1}{\sqrt{ 2 }} ≈ 0.707$倍。
2. 用分贝表示：相对于中频段，增益下降了 3 dB。
3. 求解思路（这类题目的通用解法）
   1. 明确概念：牢记 f_H 和 f_L 的定义就是基于中频电压放大倍数幅值 |A_m| 的 0.707倍 (-3dB) 来定义的。
   2. 写出关系式：
      1. $在 f = f_H (或 f_L) 时，有：|A_u(f_H)| = |A_m| / √2 ≈ 0.707 |A_m|$
   3. 计算具体数值：
      1. $如果题目给出中频增益 |A_m| = 100，那么在 f_H 处的增益幅值就是 100 × 0.707 = 70.7。$
      2. 如果题目要求下降了多少，可以直接回答“下降到中频的0.707倍”或“下降了3dB”。
      3. 最终解题流程：$20lgA_{um}=aHz, A_{um}=x, A_{uH/L}=\frac{A_{um}}{\sqrt{ 2 }}=0.707A_{um}$, 其中 a 为稳定时的频率。

**多级放大电路的总通频带，比其中任何一级的单级放大电路的通频带都更窄。**

- 对于n级参数相同的放大电路，其总电压放大倍数为各级放大倍数的乘积：$A = A_1 × A_2 × ... × A_n$。
- 每一级都有自己的上限频率 $f_H1, f_H2, ... 和下限频率 f_L1, f_L2$, ...。
- 虽然总增益提高了，但在高频和低频段，每一级的放大倍数都在下降。任何一级的增益开始显著下降，都会导致总增益更早地达到-3dB的下降点。
- **总的上限截止频率$f_H'$**与**单级上限截止频率$f_H$** 的关系为：$f_H' ≈ 0.64 f_H$
- **总的下限截止频率 $f_L'$**与**单级下限截止频率$f_L$** 的关系为：$f_L' ≈ f_L / 0.64$

增益-带宽积 (Gain-Bandwidth Product)

- 对于一个特定的放大电路（尤其是集成运放和单级放大电路），其中频增益 $|A_m|$ 和通频带 $BW$ 的乘积近似为一个**常数**。
- **公式**：$|A_m| \times BW \approx \text{常数}$
- **重要推论**：
- **增益越高，通频带越窄**。
- 要拓宽通频带，往往需要以牺牲电压放大倍数为代价。

电压放大倍数的分贝表示

- 电压增益的分贝值：$20\lg(|A_u|)$
- **中频时**：$G_m (\text{dB}) = 20\lg|A_m|$
- **在 $f_H$ (或 $f_L$) 时**：
  $G (\text{dB}) = 20\lg(|A_m|/\sqrt{2}) = 20\lg|A_m| + 20\lg(1/\sqrt{2}) = G_m (\text{dB}) + 20\lg(1/\sqrt{2})$
  因为 $20\lg(1/\sqrt{2}) = 20 \times (-0.15) \approx -3 \text{ dB}$，所以 $G (\text{dB}) = G_m (\text{dB}) - 3 \text{ dB}$。

## 反馈电路

### 判断

正/负反馈：瞬时极性法，看反馈叠加后输入是增加还是减少。
像电流一样，顺着信号流向（输入 → 放大 → 输出 → 反馈 → 返回输入端），逐级判断极性。

1. 在输入端，假设一个电压瞬时增加（用“⊕”表示）。
2. 沿着信号流向，逐级判断各级输出的瞬时极性（共射/共源电路反相，共集/共基/运放同相端同相，运放反相端反相）。
3. 看反馈信号送回到输入端后，是**削弱**了原始输入信号（净输入减小），还是**增强**了它。
   电压/电流反馈：短接输出后无反馈，电压；否则电流
   串联/并联反馈：输入短路法，看反馈信号是否还能作用

- 假想将输入端对地短路（即令 Vi=0Vi=0）。
  - 观察此时反馈信号是否还能作用到放大电路中：
    - 如果反馈信号**无法作用**到放大电路（净输入信号不受影响）→ **并联反馈**
      - _解释_：输入端短路后，反馈电流被短路点旁路到地，无法进入放大电路。
    - 如果反馈信号**仍然能作用**到放大电路 → **串联反馈**
      - _解释_：输入端短路后，反馈电压仍然能加在放大电路的输入回路上（如运放的差模输入两端）。

更快的邪道方法：
电压/电流反馈：反馈网络直接连在输出端，也就是输出直接接地，电压；串联在反馈组态里，电流
串联/并联反馈：输入信号和反馈信号不在**同一个节点**上，串联；在，并联。

### 虚短和虚断

1. **虚短：** 理想运放的两个输入端（同相输入端 `V+` 和反相输入端 `V-`）之间的电压差为零。
   - **公式： V+ = V-**
   - **原因：** 运放的开环增益为无穷大 (Avo = ∞)，而输出电压是一个有限值。根据 Vout = Avo * (V+ - V-)，要得到一个有限的 Vout，唯一的可能就是 (V+ - V-) 无限接近于零。
2. **虚断：** 流入理想运放两个输入端的电流为零。
   - **公式： I+ = I- = 0**
   - **原因：** 理想运放的输入电阻为无穷大 (Rin = ∞)，相当于断路，所以没有电流能够流进去。

\Delta U_{O1} = -\frac{\beta R_{c1}}{R_{s1}+r_{be1}}\Delta U_{I1}, \quad \Delta U_{O2} = -\frac{\beta R_{c2}}{R_{s2}+r_{be2}}\Delta U_{I2}
$$

#### 石英晶体振荡器

| 特性维度                  | RC振荡电路                                                       | 石英晶体振荡器                                                               |
| :------------------------ | :--------------------------------------------------------------- | :--------------------------------------------------------------------------- |
| **核心原理**        | 利用电阻(R)和电容(C)的充放电时间常数决定频率。                   | 利用石英晶体的**压电效应**和其固有的高Q值机械谐振。                    |
| **频率精度**        | **低**（通常为 ±1% 到 ±5%），离散性大。                  | **极高**（通常为 ±10ppm 到 ±100ppm，1ppm=0.0001%）。                 |
| **频率稳定度**      | **差**。受温度、电源电压、元件老化影响非常大。             | **极好**。温度稳定性好（特别是带温补的TCXO），几乎不受电压、老化影响。 |
| **频率可调性**      | **高**。通过改变R或C的值，可以方便地在较大范围内调节频率。 | **低**。出厂时频率基本固定，只能在极窄范围内微调（用于校准）。         |
| **输出波形**        | 通常为正弦波或非理想方波（含较多谐波）。                         | 纯净的正弦波或经过整形的标准方波。                                           |
| **电路复杂度**      | **简单**。元件少，易于集成。                               | **相对复杂**。需要外接晶体和匹配电容，或使用封装好的振荡器模块。       |
| **成本**            | **极低**。                                                 | **较高**（尤其是高精度型号）。                                         |
| **工作频率范围**    | 低频到中低频（通常几Hz到几MHz）。                                | 范围广，从 kHz 到数百 MHz（基频或泛音）。                                    |
| **启动时间**        | 快。                                                             | 相对较慢（需要建立机械振动）。                                               |
| **功耗**            | 可以做到很低。                                                   | 通常高于同频率的RC电路。                                                     |
| **抗干扰性**        | 较差，容易受外界干扰导致频率漂移。                               | 极好，对外界电磁干扰不敏感。                                                 |
| **Q值（品质因数）** | 低（通常 < 100）。                                               | **极高**（通常 > 10,000）。这是其高稳定性的物理根源。                  |
