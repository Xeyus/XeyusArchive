## About This Archive

Welcome to the Limbus Archive — a digital repository built with the philosophy that design should serve both aesthetics and accessibility.

### The Vision

This archive represents the intersection of **brutalist web design** and **modern accessibility standards**. Every pixel, every line, every interaction has been carefully considered to create an experience that is:

- **Uncompromisingly direct**: No unnecessary decorations or distractions
- **Readable above all**: Optimized typography and spacing for long-form content
- **Accessible by design**: Color-blind friendly, keyboard navigable, screen reader optimized
- **Atmospherically authentic**: CRT-inspired visual language that evokes institutional computing

### Design Principles

The theme adheres to strict design constraints:

1. **Zero rounded corners**: All boundaries are hard-edged
2. **Physical UI feedback**: Embossed/debossed states for tactile interaction
3. **Low-contrast palette**: Reduces eye strain during extended reading
4. **Semantic structure**: Clean HTML that works for everyone

### Technical Details

Built with:
- Hugo static site generator
- Pure CSS (no frameworks)
- Minimal JavaScript (only for TOC enhancement)
- Web fonts from Google Fonts
- Semantic HTML5 markup
